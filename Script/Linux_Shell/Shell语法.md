- 补充：在 Bash 版本 3 以上，在 for 循环的 in 后面，可以直接通过 {1..5} 更简洁地产生自 1 到 5 的数字（注意，1 和 5 之间只有两个点），
例如：

for i in {1..5}; do echo -n "$i "; done
1 2 3 4 5
